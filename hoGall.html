HOGALL 포트폴리오

  • 프로젝트명 : hogall
  • 수행기간 : 2019-06-20 ~ 2019-09-17
  • 개발목표 : 중고로 물건을 판매할때 물건의 가격을 책정할 때 너무높게 책정하면 판매가 되지 않고 낮게 책정하면 손해본듯한 느낌이 드는데 그것을 해결할 수 있게 경매 시스템을 사용한다.
  • 개발환경 : STS, SQLdeveloper
  • [ Front ] - HTML, CSS, JavaScript, EL, JSTL, AJAX
  • [ Back ] - Java, Jsp / Servlet
  • [ Server ] - Tomcat 8.5 Version
  • 참여도 : 100%
  • 기여도 : 100%

ERD

  • [1] 로그인은 사용자가 입력한 값을 DB에서 비교해 일치하면 session에 사용자 정보를 저장한다. 비교할 땐 사용자가 입력한 값과 암호화처리돼 저장된 비밀번호와 비교해주는 BCrypt를 사용한다.
  • [2] 검색을 할수있다. 입력을 할때마다 ajax가 발동돼 일치하는 단어가 있는 상품을 찾아준다
  • [3] 인기있는 경매물품은 조회수 순으로 정렬한다.
  • 회원가입을 할 수 있다.
  • 사용자한테 입력받은 정보를 받아 DB에 저장한다.
  • [1] 아이디는 입력할때마다 Ajax 이용해 중복이 되는 아이디가 있는지 검사함
  • [2] 비밀번호는 정규식 처리를 해 영문,숫자 조합의 8~20자 내외로만 입력 가능하게 한다. 전송 할 때 비밀번호는 암호화 처리해서 DB에 저장한다
  • [3] JavaMailSender이용해 설정에둔 발신자로 사용자가 입력한 이메일로 인증번호를 발송한다.
  • [4] 다음 우편번호 서비스 api를 이용해서 우편번호,주소를 입력하게 함.
  • [1] 상품을 등록할 수 있음 이미지는 최대 15장 업로드 할 수 있음
  • [2] 즉시구매여부를 결정할 수 있고
  • [3] 안전거래를 체크하면 낙찰시 가격이 증가되는데 그 비용을 누가 부담하는지 정할 수 있음
  • [1] 상품등록페이지에서 업로드한 사진은 슬라이드 형식으로 볼 수 있음
  • [2] 입찰자 목록은 modal창 형식으로 볼 수 있음
  • [3] 입찰하기를 누르면 현재 경매가에 입찰단위를 더해 입찰이됨.
  • [4] 댓글과 답글을 작성할수 있음.
  • 마이페이지에선 자신의 관심 카테고리를 등록,삭제를 할 수 있고 자신이 입찰한 모든 경매, 낙찰된 경매를 볼 수 있음
  • 그리고 자신의 회원정보 변경을 할 수 있음
